# kanban-bazy-danych

Projekt na przedmiot "Bazy danych" - narzędzie do zarządzania tablicami kanban

Technologie:
* Baza danych - MongoDB
* Frontend - JavaScript/React
* Serwer i API - JavaScript/Express

* stworzenie frontendu aplikacji
